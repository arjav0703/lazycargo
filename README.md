## Installation
[![Crates.io](https://img.shields.io/crates/v/lazycargo.svg)](https://crates.io/crates/lazycargo)

#### Install via Cargo (Rust's package manager)

If you have Cargo installed, you can easily install cargofetch by running the following command:
```bash
cargo install lazycargo
```

#### Manual:
You can download the pre-built binary for your platform from the [releases page](https://github.com/arjav0703/lazycargo/releases).

#### Build from source
```bash
git clone https://github.com/arjav0703/cargofetch.git
cd cargofetch
cargo build --release
```
The binary will be located in the target/release/ directory.
